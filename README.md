# PeachMate

is a _ReactJS + NodeJS_ application that allows to quickly schedule posts to be published on LinkedIn.

# Notes

## General

- Dockerised
- Makefile

## Client

- ReactJS Front-end
- Unit & Integration Tests
- - Jest
- - Storybook
- Bulma CSS Framework
- SCSS Preprocessing
- Custom Bulma Theming
- Dark Mode

## Server

- NodeJS Rest API with Express
- TDD
- - Mocha
- - Chai + Chai HTTP
- Mongoose
- Redis
- Helmet, Cors, Bull, DotEnv
